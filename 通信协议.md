# **通信功能描述**

## **基本格式**

| 序号   | 1     | 2    | 3    | 4    | 5        |
| ------ | ----- | ---- | ---- | ---- | -------- |
| 字节数 | 2     | 1    | 1    | N    | 1        |
| 名称   | FRAME | LEN  | CMD  | DATA | CHECKSUM |

 

 

***\*注解：\****

| 序号 | 名称     | 意义      | 备注                          |
| ---- | -------- | --------- | ----------------------------- |
| 1    | FRAME    | 起始标示  | 55 BB                         |
| 2    | LEN      | 数据N长度 | 数据长度：0-FF ,0：无数据信息 |
| 3    | CMD      | 命令标示  | 命令标示                      |
| 4    | DATA     | 数据N     |                               |
| 5    | CHECKSUM | 校验码    | 和校验                        |

 

明确下：是否LEN=CMD+DATA(本协议LEN不包括CMD，只是DATA的长度)

Checksum表示从帧头55BB到dataN所有数据的累和并取低八位,

Checksum = (0X55 + 0XBB + LEN + CMD + DATA + CHANNEL) & 0X0FF;

## **通信方式**

波特率为115200，一个停止位，八个数据位，无奇偶校验

***\*数据传输规则\****

单字节（8位）数据传输：低有效位在前。

多字节（>8位）数据传输：先传高字节，后传低字节。

***\*传输机制\****

采用请求应答机制，除非特殊说明，下位机每接收到一命令包后必须返回数据包

## 1. **下位机上报设备状态（CMD=0X01）**

| 序号 | 数据描述 | 字节长度 | 数据定义  | 备注                        |
| ---- | -------- | -------- | --------- | --------------------------- |
|      |          | 1        | 0x00-0x02 | 0x00:待机0x01:运行0x02:暂停 |

例：55 BB 01 01 01 13

## 2. **下位机上报设备当前选定通道号****（CMD=0X02）**

| 序号 | 数据描述 | 字节长度 | 数据定义    | 备注                  |
| ---- | -------- | -------- | ----------- | --------------------- |
|      |          | 1        | 0X00-0X01： | 0X00:通道1 0X02:通道2 |

例：55 BB 01 02 01 14

## 3. **通道选择****（CMD=0X03）**

| 序号 | 数据描述 | 字节长度 | 数据定义    | 备注                  |
| ---- | -------- | -------- | ----------- | --------------------- |
|      |          | 1        | 0X00-0X01： | 0X00:通道1 0X01:通道2 |

例：55 BB 01 03 01 15

## 4. **设置****工作时间****（CMD=0X04）**

| 序号 | 数据描述 | 字节长度 | 数据定义      | 备注      |
| ---- | -------- | -------- | ------------- | --------- |
|      |          | 2        | 0X0000-0X0258 | 0-600分钟 |

例：55 BB 02 04 00 01 17

## 5. **设置****电刺启动/停止****（CMD=0X05）**

| 序号 | 数据描述 | 字节长度 | 数据定义  | 备注                  |
| ---- | -------- | -------- | --------- | --------------------- |
|      |          | 1        | 0x00-0x01 | 0X00:启动   0X01:停止 |

 

设置电刺启动/停止

例：55 BB 01 05 01 17

 

 

## 6. **设置****电刺激模式****（CMD=0X06）**

| 序号 | 数据描述 | 字节长度 | 数据定义    | 备注                           |
| ---- | -------- | -------- | ----------- | ------------------------------ |
|      |          | 1        | 0X00-0X02： | 0x00：直流0x01: 矩形0x02: 双频 |

 

例：55 BB 01 06 01 18

 

 

## 7. **设置电刺激基波频率****（CMD=0X07）**

| 序号 | 数据描述 | 字节长度 | 数据定义        | 备注    |
| ---- | -------- | -------- | --------------- | ------- |
|      | 不进1    | 2        | 0X0000-0X2710： | 0-10KHz |

 

例：55 BB 02 07 00 01 1A

## 8. **设置电刺激基波脉宽（CMD=0X08）**

| 序号 | 数据描述 | 字节长度 | 数据定义     | 备注      |
| ---- | -------- | -------- | ------------ | --------- |
|      | 步进10   | 2        | 0X01F4-0X3E8 | 50-1000us |

设置1路启停

例：55 BB 02 08 00 01 1B

 

 

## 9. **设置电刺激电流强度 （CMD=0X09）**

| 序号 | 数据描述             | 字节长度 | 数据定义        | 备注     |
| ---- | -------------------- | -------- | --------------- | -------- |
|      | 步进0.1  数据*10发送 | 2        | 0X0000-0X0320： | 0.1-80mA |

 

设置2路导入模式

例：55 BB 02 09 00 01 1C

|      | 步进0.1  数据*10发送 | 2        | 0X0000-0X0258： | 0.1-60s |

 

例：55 BB 02 0A 00 01 1D

 

 

## 11. **设置电刺激调制波保持时间 （CMD=0X0b）**

设置2路强度
